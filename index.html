Descripción del proyecto

Al principio no teníamos idea de que era lo queríamos crear, pero todos íbamos con la idea de crear una página web, como tema surgió una página para jugadores de todo el mundo, pensamos hacer un foro, pero luego decidimos crear una página para informar de las actualizaciones de las consolas de videos juegos y poder comunicarse con otros usuarios. La idea le surgió a Ayelén y el aporte del grupo hizo extender el proyecto. El equipo fue acompañando con ingeniosas ideas con respecto al desarrollo de la esquematización y funcionalidad de la página web. La página está destinada a todas las personas apasionadas que se encuentran sumergidas en el mundo de los videos juegos.

El equipo

Arrancamos aprendiendo html y CSS, con el maquetado de la página web y eligiendo el nombre, también aprendimos jquery, de ahí surgió WORLD GAMER, luego le incorporamos las imágenes y video, utilizamos bootstrap, con este creamos menús y adaptamos todo para que esta generación del siglo XXI tenga facilidad a la hora de visualizar nuestra página, ya que todos usan celulares, computadoras o tablets. También conectamos la página web con una base de datos con Xampp. Usamos apache, mysql y php.
